pillow

Voice guardrails for coding agents.
Stays quiet — speaks up when it matters.

View on GitHub
🔊

Narrated alerts

Speaks out loud when Claude attempts dangerous or irreversible operations.

🧭

Drift detection

Catches when the agent wanders from your original goal mid-session.

📋

Session recap

Summarises what happened and how much the session cost when you're done.

✋

Slap to interrupt

Physically slap your Mac to pause the agent. Apple Silicon only.

🔒

Privacy modes

Choose cloud, hybrid, or fully local — nothing leaves your machine in local mode.

🤫

Silent by default

Only speaks on high-signal events. No noise while things are going well.

Quick install

$ brew install AMOORCHING/pillow/pillow-cli $ pillow setup $ bash plugin/install.sh $ pillow
